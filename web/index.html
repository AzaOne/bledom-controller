menu_book Docs & API

api Lua API

Create custom animations by writing Lua scripts. The agent exposes these global functions:

  • set_power(boolean) — Turns the LEDs on or off. Example: set_power(true)
  • set_color(r, g, b) — Sets the static color (0–255). Example: set_color(255, 0, 100)
  • set_brightness(value) — Sets brightness (1–100). Example: set_brightness(75)
  • sleep(milliseconds) — Pauses the script. Example: sleep(50)
  • should_stop() — (Pattern files only) Returns true if the user clicked "Stop". Use in loops: if should_stop() then return end
  • print(message) — Prints to the agent console log for debugging.

animation Lua Effects

These functions perform complex animations and block until complete or cancelled.

  • breathe(duration_ms) — Smooth 1%→100% brightness pulse. Example: set_color(0,255,255); breathe(4000)
  • strobe(r, g, b, duration_ms, freq_hz) — Flash for a total duration. Example: strobe(255,255,255, 5000, 10)
  • fade(r1,g1,b1, r2,g2,b2, duration_ms) — Cross-fade between colors. Example: fade(255,0,0, 0,0,255, 3000)
  • fade_brightness(start, end, duration_ms) — Smooth brightness transition.

schedule Cron Scheduler

Uses standard cron syntax to automate commands.

  • Syntax: MINUTE HOUR DOM MONTH DOW.  Example: 0 22 * * 1-5 = 10 PM Mon–Fri. (crontab.guru)
  • Commands:
    • power on / power off
    • pattern [filename.lua] — Run a full pattern file
    • lua[lua_code] — Execute a single line of Lua
